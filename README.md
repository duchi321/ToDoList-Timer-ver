## Features
* 顯示當前時間點。
* 產生的每筆資料都會擁有計時功能。
* 完成的資料有特定顯示。
* 使用localstorage,可記錄資料內容。
* 光與暗2種風格模式可切換。
---
## How to use
1. 輸入文字後，點選Add或按下Enter,就會登入項目並開始計時。
2. 對Todo區域文本按下滑鼠左鍵,可轉移項目到Done區域，表示完成項目。
3. 叉叉按鈕『X』，表示清除該項目。
4. Add按鍵旁的切換紐可以轉換頁面風格。
